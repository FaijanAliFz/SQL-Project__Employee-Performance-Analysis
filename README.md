# 📊 Employee Performance Analysis (SQL)

---

## 📊 Project Overview
This project analyzes employee performance, attendance, and sales data using SQL. The goal is to evaluate individual and departmental performance, identify key contributors, and uncover trends that support better workforce management.

---

## 🎯 Objectives
- Identify top-performing employees and managers  
- Analyze attendance patterns and absenteeism  
- Evaluate department-wise performance  
- Assess sales contribution across employees  
- Detect low-performing and high-performing employees  

---

## ⚙️ Key Analysis Performed
- Employee-wise sales and performance ranking  
- Manager-level performance comparison  
- Attendance rate and absenteeism analysis  
- Bonus distribution analysis  
- Department-wise performance trends  
- Identification of employees above/below average performance  

---

## 🛠️ SQL Techniques Used
- Joins (Inner Join, Left Join)  
- Aggregations (SUM, AVG, COUNT)  
- Window Functions (RANK, DENSE_RANK)  
- Common Table Expressions (CTE)  
- Subqueries  
- Group By & Order By  

---

## 📈 Key Insights
- Identified top-performing employees based on sales and attendance  
- Highlighted employees with high absenteeism  
- Compared manager performance across teams  
- Analyzed department-wise productivity differences  
- Evaluated bonus allocation patterns  

---

## 🚀 Project Outcome
Converted raw employee data into **structured performance insights**, enabling better decision-making in employee management, productivity improvement, and organizational planning.
